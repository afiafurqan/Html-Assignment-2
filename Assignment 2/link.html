Absolute Url's

HTML
HTML provides the structure of a webpage (like headings, paragraphs, images, links, lists, etc.). It is not a programming language; its a markup language that tells the browser how to display content.
CSS
CSS (Cascading Style Sheets) is a stylesheet language used to describe how HTML elements should look on a webpage. It controls the design aspects such as layout, colors, fonts, spacing, and responsiveness, making websites more attractive and user-friendly.
JavaScript
JavaScript is a programming language used to make webpages interactive and dynamic. It allows you to add features like form validation, animations, pop-ups, and real-time updates.
